# Unit 004 計画: README再構成

## 概要

README.mdをユーザー向けクイックリファレンスとして再構成し、docs/README.mdとの重複を解消する。

## 変更対象ファイル

- `README.md` — 4セクション構成に再構成
- `docs/README.md` — 詳細リファレンスとして位置付け、相互参照を追加

## 実装計画

### README.md 新構成

1. **インストール手順** — make install + 初回実行
2. **クイックスタート** — 基本コマンド例
3. **設定リファレンス** — config-defaults.sh の全設定項目一覧
4. **トラブルシューティング** — よくある問題と解決方法

### 重複解消方針

- README.md: 簡潔なクイックリファレンス（ユーザー向け）
- docs/README.md: アーキテクチャ・詳細仕様（開発者・詳細情報向け）
- 両ファイル間で相互参照リンクを設置
